Quick Start

Contribute with a new package or new service

Create a new package
  1. Create your folder /PACKAGE/
  2. Create your first service in /packages/PACKAGE/src/SERVICE.ts

For example a service Storage :

export default class Storage<T extends StorageParameters = StorageParameters> extends Binary<T> {

}
  1. Add your service name in /packages/PACKAGE/webda.module.json targeting final /lib/ files
{
  "services": {
    "Webda/GoogleCloudStore": "./lib/services/storage.js"
  }
}
  1. Create your first unit test in /packages/PACKAGE/src/SERVICE.spec.ts
  2. Define your new service in /packages/PACKAGE/test/config.json If you extends Binary, you need to copy Users service frlom existing other package.
